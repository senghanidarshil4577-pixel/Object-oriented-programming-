About the Project

What is this system?

The Student Management System is a console-based application designed to simplify how educational institutions handle student data. It allows administrators to add, view, search, update, and delete student records with ease.

The core application is written in C++ using the principles of Object-Oriented Programming — keeping the codebase modular, maintainable, and scalable. This web interface showcases its design and key concepts interactively.

Core Concepts

OOP Pillars

The four fundamental principles of Object-Oriented Programming applied in this project.

⬡

Encapsulation

Student data and methods are bundled inside a class, hiding internal state from outside interference and exposing only necessary interfaces.

class Student { private: int id; ... };
◈

Inheritance

Student inherits shared attributes like name and age from the base Person class, promoting code reuse and a clean hierarchy.

class Student : public Person { ... };
◎

Abstraction

Complex implementation details are hidden behind simple interfaces. Users interact with high-level methods without needing to know the internals.

student.display(); // hides logic
⬠

Polymorphism

The same function name behaves differently depending on context — virtual functions allow derived classes to override base class behavior.

virtual void display() override;
